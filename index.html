narodmon.ru клиент для android.

Статус

На данный момент программа находится в активном бэта-тестировании.

Платформы

Поддерживаются android 2.3 и новее.

Функционал

Основной экран приложения отображает список датчиков. По кнопке "Menu" вызывается меню с пунктами:

  • Обновить - обновляет список и показания по всем датчикам
  • Авторизация - вызов диалога авторизации
  • Настройки - настройки программы.

В правом верхнем углу есть кнопка, она показывает диалог с фильтрами и сортировкой.

Есть второй экран со списком датчиков - "Выбранные" изначально он пустой. Переключаться между экранами можно выбрав экран из списка (выпадающий список находится у иконки приложения в левом верхнем углу) или горизонтальным движением пальца по экрану.

Тап по датчику открывает экран с подробной информацией и графиком изменения параметра. На этом экране есть кнопка в правом верхнем углу, с пиктограммой "глаз", нажатие на нее меняет ее цвет. Если она голубая - датчик добавлен в список "Выбранные". Убрать датчик из списка можно так же, нажав на кнопку, при этом она станет серой.

Когда датчик добавлен в "Выбранные" под иконкой в виде глаза появляется иконка с пиктограммой "колокольчик". Это кнопка вызывает диалог установки оповещения, если значение параметра датчика пересечет один из порогов - сработает вибрация.

Можно настроить следующий тип контроля:

  • Превышение порога
  • Снижение ниже порога
  • Выход за диапазон из двух порогов
  • Вход в диапазон из двух порогов

Далее следует изменить верхний и нижний пороги и нажать кнопку "Установить". Выключить оповещение можно, установив тип контроля в "НЕТ" и нажав "Установить".
